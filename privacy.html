隐私政策

最后更新日期：2026年6月16日

我们收集什么信息

什么都不收集。

本应用（"视频发布助手"）不收集、不存储、不传输任何个人信息。具体来说：

  • ❌ 不收集姓名、邮箱、电话号码
  • ❌ 不收集位置信息
  • ❌ 不收集设备信息
  • ❌ 不使用 Cookie 或追踪技术
  • ❌ 不分析用户行为

TikTok API 数据

本应用通过 TikTok 官方 API 进行操作。所有数据交互发生在：

  • 你的本地电脑 ↔ TikTok 服务器

本应用仅作为"遥控器"，在中间传递指令，不保存任何数据副本。

Token 存储

OAuth 访问令牌（Access Token）仅存储在你的本地电脑上（~/.claude/tiktok-uploader/config.json），不会上传到任何服务器。

第三方服务

本应用不使用任何第三方分析、广告或统计服务。

数据安全

所有 API 通信通过 HTTPS 加密传输。本地存储的令牌文件权限设置为仅用户本人可读取。

儿童隐私

本应用不面向 13 岁以下儿童，不收集任何人的任何信息。

政策更新

如果本政策有变更，会在此页面更新。
